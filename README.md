# 基于Spark的学生完成学业情况数据集可视化分析与预测

## 简介
采用Spark对Kaggle的学生数据集进行处理、分析与计算，基于随机森林决策树朴素贝叶斯决策树四种算法构建四种模型，将分析结果与模型上传至HDFS，使用Python和Echarts对数据可视化。
## 技术栈
Spark + HDFS + Python + Echarts
